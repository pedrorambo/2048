# 2048

## Compilar e rodar o projeto
Dê permissão de execução ao arquivo `run.sh` com `chmod +x run.sh`.

Execute `./run.sh`.

## Dependências

### Ncurses
Comando para instalação com apt-get: `sudo apt-get install libncurses5-dev libncursesw5-dev`
